# 🎓 Python 知识测验平台

一个现代、交互式的 Python 编程知识测验平台，帮助开发者巩固和检验 Python 基础知识。

## 📋 项目简介

这是一个前端应用，提供了一个用户友好的界面来进行 Python 编程知识测验。平台包含 20 道精心设计的题目，涵盖 Python 的基础概念、数据类型、函数、面向对象编程等多个方面。

## ✨ 主要特性

- 🎯 **20 道精选题目** - 涵盖 Python 基础到中高级知识点
- 📊 **实时反馈** - 提交答案后立即显示正确/错误
- 📚 **详细解析** - 每道题都包含答案解析和知识点说明
- 📈 **进度跟踪** - 可视化进度条显示答题进度
- 💾 **成绩统计** - 记录测验成绩和统计数据
- 🎨 **现代化设计** - 响应式设计，支持各种设备
- 🌙 **友好的用户界面** - 清晰的视觉层次和流畅的交互

## 📖 使用说明

### 主页 (index.html)

- 显示平台欢迎信息
- 展示平台统计数据（总题数、平均分数、已完成数量）
- 提供"开始答题"按钮进入答题页面
- 包含使用指南

### 答题页面 (quiz.html)

1. **题目加载** - 每页显示一道题目
2. **选项选择** - 从 4 个选项中选择正确答案
3. **答案提交** - 点击"提交答案"按钮
4. **即时反馈** - 显示答案是否正确
5. **答案解析** - 提供详细的解答说明
6. **继续答题** - 导航到下一题或返回主页

## 🔧 核心功能详解

### 1. 题目管理 (data/questions.js)

每道题包含：
- `id` - 题目编号
- `title` - 题目标题
- `description` - 题目描述
- `options` - 4 个选项
- `correctAnswer` - 正确答案索引（0-3）
- `explanation` - 答案解析
- `difficulty` - 难度等级（easy/medium/hard）
- `category` - 知识分类

### 2. 判分逻辑 (assets/js/quiz.js)

- 跟踪用户的每个答案
- 记录已答题的题目
- 计算最终分数（每题 10 分）
- 显示答题统计

### 3. 样式系统 (assets/css/style.css)

- CSS 变量定义全局样式
- 卡片、按钮、选项等组件样式
- 响应式断点设置
- 动画和过渡效果
